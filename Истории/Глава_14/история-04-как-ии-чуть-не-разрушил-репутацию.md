# История: Как ИИ чуть не разрушил репутацию

**Контекст**: Глава 14  
**Стиль**: Предостережение ⚠️

—-

**Инцидент: AI-generated код с ошибкой**

*Действия:*
1. Коллега написал статью с ChatGPT
2. Не проверил код
3. Опубликовал на Habr

*Результат через 2 часа:*
- Комментарии: "Код не работает!"
- "Даже не запустили перед публикацией?"
- "Это ChatGPT писал, да? Видно сразу"

**Ущерб репутации:**

```
Доверие: 100% → 60% (один инцидент)
Время восстановления: 3-6 месяцев
Стоимость: потеря позиции эксперта
```

**Что пошло не так:**

❌ Слепое доверие ИИ  
❌ Нет проверки кода  
❌ Нет review перед публикацией

**Правильный процесс:**

✅ ИИ генерирует черновик  
✅ Человек проверяет факты  
✅ Код тестируется  
✅ Peer review  
✅ Публикация

**Вывод:** ИИ - инструмент, не замена мышлению. Как молоток: правильно используешь - забиваешь гвоздь, неправильно - по пальцу.
