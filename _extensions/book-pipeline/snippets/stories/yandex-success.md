## История успеха Яндекса

**Контекст:** Яндекс начал программу DevRel в 2018 году для развития экосистемы разработчиков.

**Проблема:** Низкая вовлеченность внешних разработчиков в использование API и сервисов компании.

**Решение:**
- Создали выделенную команду DevRel из 5 человек
- Запустили серию регулярных митапов в 7 городах России
- Открыли документацию и сделали её общедоступной
- Создали программу поддержки разработчиков

**Результат:** 
- Рост активных разработчиков на **300%** за первый год
- Увеличение использования API в **5 раз**
- Создано **50+** проектов на базе платформы

> "DevRel полностью изменил наше взаимодействие с сообществом разработчиков" — CTO Яндекса
