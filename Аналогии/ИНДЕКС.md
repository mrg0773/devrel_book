# Индекс аналогий и метафор

Этот файл содержит список всех аналогий, метафор и сравнений из книги "База DevRel".

## Типы аналогий

- **Бабушка-тест**: Объяснение сложных концепций простыми словами
- **Температурные аналогии**: Сравнение с измерением температуры
- **Метафоры**: Образные сравнения
- **Спортивные аналогии**: Сравнения со спортом
- **Другие аналогии**: Различные сравнения для понимания

## Формат плейсхолдера в тексте

В тексте главы аналогия заменяется на:
```
{{АНАЛОГИЯ: глава_XX_название}}
```
