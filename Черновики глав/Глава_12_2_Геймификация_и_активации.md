## Метрики эффективности и <TERM:РОИ>>

**Вы наверное думаете:** «Опять эти метрики! Я же не математик.» Не переживайте! Мы разберем все простыми словами. Это как считать деньги в кошельке — не сложно, но важно.

### Формулы расчета <TERM:РОИ>>

#### Базовая формула <TERM:РОИ>>
```

ROI = (Доходы — Затраты) / Затраты  x  100>>
```

**Простыми словами:** Если потратили 100 рублей, а получили 300 — <TERM:РОИ>> = 200>>. То есть заработали в 2 раза больше, чем потратили.

#### Детальная формула для геймификации
```

ROI = (Ценность_лидов + Ценность_контента + Ценность_бренда + Ценность_найма) / Затраты  x  100>>

Где:
— Ценность_лидов = Количество_лидов  x  Средняя_стоимость_лида  x  Конверсия_в_клиента

— Ценность_контента = Количество_контента  x  Стоимость_создания_контента

— Ценность_бренда = Увеличение_узнаваемости  x  Средний <TERM:ltv>>_клиента

— Ценность_найма = Количество_нанятых  x  Экономия_на_рекрутинге
```

### Метрики вовлечения с формулами

#### 1. Партикипашн Уровень (Уровень участия)
```

Уровень_участия = (Участники_активности / Общее_количество_посетителей_стенда)  x  100>>

Бенчмарки:
— Плохо: < 20>> (как будто никто не пришел)

— Хорошо: 20-40>> (нормальная активность)

— Отлично: > 40>> (все хотят участвовать!)

**Метафора из жизни:** Это как с рестораном — если из 100 прохожих заходят только 5, что-то не так. А если 30-40 — значит, все в порядке.
```

#### 2. Время вовлечения
```

Среднее_время_у_стенда = Общее_время_всех_участников / Количество_участников

Бенчмарки:
— Без геймификации: 1-3 минуты (взял флаер и ушел)

— С геймификацией: 10-30 минут (заигрался и забыл про время)

— Цель: увеличение в 5-10 раз

**Почему это важно:** Чем дольше человек у стенда, тем больше шансов, что он запомнит вашу компанию и захочет работать с вами.
```

#### 3. Повторное вовлечение
```

Уровень_повторения = (Количество_возвращений / Общее_количество_участников)  x  100>>

Бенчмарки:
— Хорошо: 20>>+

— Отлично: 40>>+
```

#### 4. Социальные Амплификашн (Социальное усиление)
```

Социальное_усиление = (Количество ПГК_постов  x  Средний_охват_поста) / Количество_участников

Где пользовательский контент = Пользователь Джинуровень Контент (пользовательский контент)
```

### Метрики качества <TERM:lead_множественное_родительный>>

#### <TERM:lead>> Качество Балл (Оценка качества <TERM:lead_винительный>>)
```

Балл_качества_лида = (Технический_уровень  x  0.3) + (Заинтересованность  x  0.3) + (Соответствие_вакансиям  x  0.4)

Где каждая метрика оценивается от 1 до 10:
— Технический_уровень: на основе сложности решенных задач

— Заинтересованность: время у стенда + глубина вопросов

— Соответствие_вакансиям: релевантность опыта и навыков
```

#### <TERM:conversion>> Фуннел (<TERM:funnel>> <TERM:conversion_дательный>>)
```

Этап_1: Посетители_стенда  ->  Участники_активности (Партикипашн Уровень)

Этап_2: Участники_активности  ->  Качественные_лиды (Лид Качество Уровень)

Этап_3: Качественные_лиды  ->  Созвоны/интервью (интервью Уровень)

Этап_4: Созвоны  ->  Предложения_о_работе (Оффер Уровень)

Этап_5: Предложения  ->  Принятые_предложения (Найм Уровень)

Общая конверсия = Этап_1  x  Этап_2  x  Этап_3  x  Этап_4  x  Этап_5
```

### Примеры расчетов с реальными цифрами

#### <TERM:case_study>>: Технический квиз на <TERM:conference_дательный>>

**Исходные данные:**

— Затраты: 30К рублей

— Участники: 150 человек

— Посетители стенда: 500 человек

— Качественные <TERM:lead_множественное>>: 45 человек

— Созвоны: 15 человек

— Предложения: 5 человек

— Нанято: 2 человека

**Расчет метрик:**

```

Партикипашн Уровень = 150 / 500  x  100>> = 30>>

Лид Качество Уровень = 45 / 150  x  100>> = 30>>

интервью Уровень = 15 / 45  x  100>> = 33>>

Оффер Уровень = 5 / 15  x  100>> = 33>>

Найм Уровень = 2 / 5  x  100>> = 40>>

Общая конверсия = 0.30  x  0.30  x  0.33  x  0.33  x  0.40 = 0.39>>
```

**Расчет <TERM:РОИ>>:**

```

Ценность найма = 2  x  500 тыс. руб. (средняя стоимость найма) = 1,000 тыс. руб.

ROI = 1,000 тыс. руб. / 30 тыс. руб.  x  100>> = 3,333>>
```

#### <TERM:case_study>>: Геймифицированный квест (<TERM:budget>> 100К)

**Исходные данные:**

— Затраты: 100К рублей

— Участники: 200 человек

— <TERM:ugc>> постов: 80 постов

— Средний <TERM:reach>> поста: 500 просмотров

— Качественные <TERM:lead_множественное>>: 60 человек

— Нанято: 3 человека

**Расчет метрик:**

```

Социальные Амплификашн = (80  x  500) / 200 = 200 просмотров на участника

Ценность найма = 3  x  500 тыс. руб. = 1,500 тыс. руб.

Ценность контента = 80  x  2 тыс. руб. (стоимость создания поста) = 160 тыс. руб.

Ценность бренда = 40 тыс. руб.  x  50 тыс. руб. (<TERM:ltv>>) = 2,000 тыс. руб.

ROI = (1,500 тыс. руб. + 160 тыс. руб. + 2,000 тыс. руб.) / 100 тыс. руб.  x  100>> = 3,660>>
```

### Продвинутые метрики

#### Нет Промотер Балл (<TERM:nps>>) для активностей
```

<TERM:nps>> = >>_промоутеров — >>_критиков

Где:
— Промоутеры: оценка 9-10 («Рекомендую друзьям»)

— Критики: оценка 0-6 («Не рекомендую»)

— Нейтралы: оценка 7-8

Бенчмарки:
— Плохо: < 0

— Хорошо: 0-50

— Отлично: > 50
```

#### Кустомер Жизньвремя Валуе (<TERM:ltv>>) для <TERM:devrel>>
```

<TERM:ltv>> = (Средняя_зарплата_сотрудника  x  Время_работы_в_компании) + (Вклад_в_продукт  x  Коэффициент_влияния)

Где:
— Время_работы_в_компании = 2-3 года (среднее для <TERM:it>>)

— Вклад_в_продукт = оценка влияния на продукт (1-10)

— Коэффициент_влияния = 0.1-0.3 (зависит от роли)
```
